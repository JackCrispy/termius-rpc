# Termius RPC

## Install

1. Download ZIP
2. `npm i`
3. `npm run termius`

## How to run 24/7 in background

  - Install AutoHotKey
  - Double Click/Run the start.ahk.
  
  - Done! It is now running in the background.
  
## Notes

It will auto update every 5 seconds.
There are some errors in console.
